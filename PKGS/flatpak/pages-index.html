RSPlayer flatpak repository

Flatpak repository for the RSPlayer desktop app.

Install (adds the rsplayer remote automatically):

flatpak install https://ljufa.github.io/rsplayer-flatpak/io.github.ljufa.rsplayer.flatpakref

Or add the remote first:

flatpak remote-add --if-not-exists rsplayer https://ljufa.github.io/rsplayer-flatpak/rsplayer.flatpakrepo
flatpak install rsplayer io.github.ljufa.rsplayer

Updates arrive through flatpak update.
